A simple plugin that fetches news items from newsapi.org. Allows customers to fetch news items from their desired domains and date ranges.

== Description ==

This is a simple plugin, part of a practical test asked to be performed by CreedAlly. This plugin allows WooCommerce customers to check the latest news from their desired sources. This also creates a widget that can be used anywhere on the site.

= Special Features =
* Customer ability to check out desired news items.
* Simple widget that shows the news items based on the current logged-in customer preferences.
* Creates a log for the admin to see if the plugin is working fine or not.

== Installation ==
1. Upload the entire `creedally-api-practical` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the **Plugins** screen (**Plugins > Installed Plugins**).

== Frequently Asked Questions ==

= How much of past news can be accessed? =

The customers can check through any past news. They just need to update their news preferences from their own dashboard.

= Do I need to pay or subscribe to any third-party service for accessing the news? =

Yes, you can start by using a free subscription from https://newsapi.org. Later if you feel the news is good and the results are as expected, you can pay the news providers too.
